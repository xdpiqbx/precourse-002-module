# Array, Matrix

```java
int[] array = new int[5];
array[0] = 1;
array[1] = 2;
array[2] = 3;
array[3] = 5;
```

```java
String[] cars = {"Camaro", "Corvette", "Tesla"}; // same data type
```

```java
int[][] aMatrix = new int[3][2];
aMatrix[0][0] = 0;
aMatrix[0][1] = 1;
aMatrix[1][0] = 2;
aMatrix[1][1] = 3;
aMatrix[2][0] = 4;
aMatrix[2][1] = 5;
```
